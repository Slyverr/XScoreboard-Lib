# Scoreboard-Lib
An easy way to create & customize spigot scoreboards

# Maven
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.slyvr.libs</groupId>
    <artifactId>scoreboard-lib</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
